# DESCRIPTION 

   STUDENT MANAGEMENT SYSTEM

This is a console-based Student Management System developed in C. The system allows users to manage student records by adding, viewing, updating, deleting, and searching for students.

# Features 

Add new student records
View all students
Update existing student information
Delete student records
Search for a student using ID

# Technologies Used

C Programming Language
GCC Compiler
Termux

# How to Run the Program

Open terminal or command prompt
Navigate to the project folder
Compile the program: gcc student_management.c -o student_management
Run the program: ./student_management

#How the System Works

The program displays a menu with different options. The user selects an option (1–6), and the system performs the selected operation. Student data is stored temporarily while the program is running.
